How Much Should I Pay Per TB for an External SSD?

In 2026: $60–$90/TB for 1TB drives (T7, Extreme), $45–$70/TB for 2TB, $35–$55/TB for 4TB. USB4/Thunderbolt drives command a $25–$60/TB premium over USB 3.2 Gen 2 equivalents — justified only if your host port supports the extra bandwidth.

The Interface Trap

Buying a USB4 SSD for a laptop that only has USB 3.2 Gen 2 ports wastes money — the drive will top out at 1,000 MB/s regardless. Check your laptop spec sheet under "USB" before choosing an interface. This tracker shows both drive speed and interface ceiling.

What Speed Do I Actually Need?

Video editing (4K H.264/H.265): 80–150 MB/s sustained — any USB 3.2 Gen 2 drive handles this fine. Video editing (8K RAW): 500–1,200 MB/s — needs USB4 or Thunderbolt. PS5 game storage (PS4 titles only): 50–250 MB/s — any USB 3.2 Gen 2 drive. General backup: USB 3.2 Gen 2 is more than sufficient.

External SSD FAQ

What is the fastest external SSD in 2026?

USB4 40Gbps and Thunderbolt 3/4 drives achieve 2,000–3,000 MB/s real throughput. OWC Envoy Pro FX (2,800 MB/s, Thunderbolt 3), Samsung T9 (2,000 MB/s, USB4 20Gbps), and Crucial X10 Pro (2,100 MB/s, Gen2x2) are the current leaders in consumer price/performance.

Is USB 3.2 Gen 2 fast enough for an external SSD?

Yes for most users. USB 3.2 Gen 2 tops at ~1,000 MB/s — enough for 4K video editing, Plex, and fast backup. You'd need USB4 or Thunderbolt only for 8K RAW, ProRes RAW, or moving 100+ GB files regularly. Check your laptop spec sheet before upgrading.

What external SSD is compatible with PS5?

PS5 needs USB 3.1 Gen 2 (10Gbps) minimum for PS4 game storage. Samsung T7, SanDisk Extreme, Crucial X8/X9, and WD My Passport SSD all qualify. Note: PS5 native games cannot run from external SSD — only PS4 games. For PS5 game storage, install M.2 NVMe internally.

Samsung T7 vs SanDisk Extreme — which to buy?

Both do ~1,050 MB/s on USB 3.2 Gen 2. Samsung T7 = lighter, more compact. SanDisk Extreme = IP55 weather resistance. For rugged use: SanDisk Extreme or Samsung T7 Shield. For everyday carry: Samsung T7 base model. Tiebreaker: current $/TB on this tracker.

Is USB4 worth the extra cost?

Only if your host port supports USB4 or Thunderbolt 3/4, AND you regularly move 50+ GB files. Samsung T9 (USB4 20Gbps) vs T7 (Gen 2): ~2× faster, ~30% more expensive. For most portable storage needs, USB 3.2 Gen 2 is plenty fast.

What is a good price per TB for external SSD?

In 2026: $60–$90/TB for 1TB, $45–$70/TB for 2TB, $35–$55/TB for 4TB. SSDs remain 3–5× more expensive per TB than internal HDDs — the premium covers speed, portability, and durability. 4TB models offer the best $/TB at scale.

Can I use an external SSD with a Mac?

Yes. All USB-C SSDs work with Mac. For Time Machine: format as APFS (Mac only) or exFAT (cross-platform). Crucial X9 Pro for Mac comes pre-formatted APFS. USB4 drives are fully compatible with Thunderbolt 3/4 MacBooks.

Are rugged external SSDs slower?

No. Rugged = same internals + protective housing. SanDisk Extreme (IP55, rugged) matches Samsung T7 (non-rugged) at ~1,050 MB/s. Samsung T7 Shield adds rubber armor with no speed penalty. Go rugged if you travel frequently or work outdoors.

USB 3.2 Gen 2 vs Gen 2x2 — what's the difference?

Gen 2 (single lane) = 10Gbps = ~1,000 MB/s. Gen 2x2 (dual lane) = 20Gbps = ~2,000 MB/s. Gen 2x2 requires a host port that supports it — rare on laptops, available on recent desktop boards. Crucial X10 Pro uses Gen 2x2 and hits ~2,100 MB/s when connected to a compatible port.
